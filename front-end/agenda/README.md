## Como reproduzir

1. Clone o repositório

2. Install as depedências
```bash
npm i
```

3. Rode o app com
```bash
npm start
```

4. Você pode, opcionalmente, construir o projeto em execução
```bash
npm run build
```
5. Por fim, accesse http://localhost:3000 no seu browser
